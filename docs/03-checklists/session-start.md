# Start Work Checklist

## 1. Confirm Project Status

- [x] Open project in VS Code.
- [x] Open Terminal in the project directory.
- [x] Run `git status`.
- [x] Pull latest changes using `git pull`.
- [x] Verify GitHub repository is connected.
- [x] Confirm GitHub Pages deployment status.
- [x] Open the live website.
- [x] Review previous session documentation.

---

## 2. Plan Today's Work

- [x] Define today's primary objective.
- [x] Review current project phase.
- [x] Break the objective into smaller tasks.
- [x] Identify files likely to change.
- [x] Determine today's stopping point.

---

## 3. Begin Development

- [x] Start Live Server (if needed).
- [x] Implement one task at a time.
- [x] Test changes frequently.
- [x] Document important decisions while working.
